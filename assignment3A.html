Introduction to HTML

HTML (Hypertext Markup Language) is used to create a document on the World Wide Web. It is a simply a collection of certain key words called "Tags" that are helpful in writing the document to be displayed using a browser on Internet. HTML was developed by Tim Berers-Lee in 1992.
